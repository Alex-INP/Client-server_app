server_main module¶

Главный модуль серверной стороны. Принимает и отправляет сообщения клиентам.

class server_main.Server(addr, port, database)¶

Bases: threading.Thread

Класс-демон, является дочерним классом по потношению к threading.Thread. Реализует логику сервера. Получает сообщения от пользователей, обрабатывает их, высылает ответы, и пересылает сообщения от одних пользователей другим.

server_main.main()¶

Главная управляющая функция серверной стороны. Осуществляет стартовую инициализацию необходимых параметров. Парсит аргументы запуска, если они есть. Проверяет валидность порта. Инициализирует базу данных и создает экземпляры классов, ответственных за интерфейс, и осуществляет запуск интерфейса. Активирует поток ответственный за обмен сообщениями и их анализ.
